# Final Project

This is the final project for the Python course. The project demonstrates the application of various Python concepts learned throughout the course.

## Features

- Utilizes Python 3.x
- Implements key programming concepts
- User-friendly interface

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ItzHipe/Final_Project
    ```
2. Install the required dependencies:
    ```bash
    pip install imageio
    ```

## Usage

Run the main script to start the application:

```bash
python final_project.py
```
